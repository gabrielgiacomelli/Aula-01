Tag h

Vão de 1 a 6, servem oara definir a estrutura, hierarquia e títulos do conteúdo em uma página web.

Tag p

Indica ao navegador onde um parágrafo começa e termina, facilitando a organização do conteúdo e a leitura.

Tag a

Conhecido como âncora, é o elemento fundamental utilizado para criar hiperlinks, conectando documentos, páginas internas/externas, arquivos ou e-mails.

Tag head

Guarda informações como o título da página, codificação de caracteres, scripts e folhas de estilo.

Tag body

É a tag que engloba todo o conteúdo visível de uma página web (textos, imagens, vídeos, links).

Tag html

Define o início e o fim do documento.

Tag title

Define o título principal de uma página web, exibido na aba do navegador e como link clicável nos resultados de busca.

Tag img

Incorpora imagens externas (JPG, PNG, GIF) em uma página HTML.

Tag div

É um contêiner genérico usado para agrupar elementos e estruturar layouts no HTML.

Tag ul

Define uma lista de itens sem ordem específica, marcada por símbolos.

Tag ol

Define uma lista numerada onde a ordem dos itens é essencial

Tag button

Define uma área clicável e interativa, utilizada para disparar ações.

Tag meta

Especifica metadados como charset, descrição e viewport.

Tag br

Quebra de linha.

Tag b

Texto em negrito

Tag strong

Texto em negrito que tem peso semântico.

Tag video

Incorpora vídeos na página, permitindo controle de reprodução como play, pause e volume.

Tag audio

Incorpora arquivos de áudio na página com controles de reprodução.

Tag source

Define múltiplas fontes de mídia para elementos como video e audio.

Tag canvas

Permite desenhar gráficos e animações usando JavaScript.

Tag svg

Define gráficos vetoriais escaláveis dentro da página HTML.

Tag main

Representa o conteúdo principal da página, excluindo cabeçalhos, rodapés e navegação.

Tag aside

Representa um conteúdo secundário ou complementar ao conteúdo principal.

Tag figure

Agrupa imagens, ilustrações ou gráficos junto com sua descrição.

Tag figcaption

Define a legenda de um elemento figure.
